# Mini Healthcare Support Web App

## Overview
A simple healthcare support web app for NGOs to collect patient queries and volunteer registrations.

## Features
- Patient/Volunteer contact form
- Conceptual AI FAQ chatbot
- Auto form handling using Netlify

## AI / Automation Idea
A basic chatbot concept to answer common FAQs.  
Future scope includes integrating an AI API for intelligent responses.

## Tech Stack
- HTML, CSS, JavaScript
- Netlify for hosting

## NGO Use Case
Helps NGOs streamline support requests, volunteer onboarding, and reduce manual effort.
## Design & UX
The UI follows a clean healthcare-oriented design with soft colors, clear hierarchy, and subtle animations to ensure accessibility and professional appeal.

## AI Integration
The chatbot integrates Google's Gemini API to provide real-time responses to healthcare and volunteering-related queries.

> Note: The API key is used on the client side for demo purposes. In production, this would be securely handled via a backend or serverless function.
