# Mini Healthcare Support Web App

## Overview
A simple healthcare support web app for NGOs to collect patient queries and volunteer registrations.

## Features
- Patient/Volunteer contact form
- Conceptual AI FAQ chatbot
- Auto form handling using Netlify

## AI / Automation Idea
A basic chatbot concept to answer common FAQs.  
Future scope includes integrating an AI API for intelligent responses.

## Tech Stack
- HTML, CSS, JavaScript
- Netlify for hosting

## NGO Use Case
Helps NGOs streamline support requests, volunteer onboarding, and reduce manual effort.
